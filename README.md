Dynamic Watchlist Application using Spring Boot, JPA, and OMDB API

WatchFlow is a user-friendly application designed to help users manage their movie and web series collections efficiently. Built using Spring Boot, JPA, and MVC architecture, the app allows users to easily add, edit, and delete items from their watchlist while tracking where to stream their favorite content.

The application is integrated with the OMDB API to automatically fetch detailed information, such as movie titles, genres, release dates, and more, ensuring that users have the latest data at their fingertips. It enhances user experience by offering a clean and intuitive interface, while the back-end is optimized for smooth performance and scalability.

Key Features:

Watchlist Management: Add, edit, and delete movies and web series with ease.
OMDB API Integration: Automatically retrieves detailed movie and series information.
Source Tracking: Allows users to track available streaming sources for each item.
User-Friendly Interface: Clean, responsive UI for a seamless browsing experience.
Efficient Back-End: Built with Spring Boot and JPA for robust performance.
Tech Stack:

Back-End: Spring Boot, JPA, MySQL
Front-End: HTML, CSS, JavaScript
API: OMDB API
Architecture: MVC (Model-View-Controller)
